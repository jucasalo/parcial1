Bienvenido a la API de Recetas

La API de Recetas te permite interactuar con una base de datos de recetas de manera sencilla y eficiente. Podrás acceder a información detallada sobre distintas recetas, crear nuevas, actualizar las existentes y filtrar por diversos criterios como categoría o dificultad. Además, la API permite gestionar usuarios, desde su registro hasta la consulta de sus datos. Cada endpoint ha sido diseñado para ofrecer la máxima funcionalidad, facilitando la integración en cualquier aplicación que necesite manejar recetas de cocina o usuarios relacionados.

Aquí puedes acceder a los diferentes endpoints disponibles:

Usuarios

  • Ver todos los usuarios: GET /usuarios
  • Registrar usuario: POST /usuarios

Recetas

  • Obtener todas las recetas: GET /recetas
  • Obtener una receta por ID: GET /recetas/:id
  • Crear una receta (requiere autenticación): POST /recetas
  • Actualizar una receta por ID (requiere autenticación): PUT /recetas/:id
  • Eliminar una receta por ID (requiere autenticación): DELETE /recetas/:id

Ingredientes

  • Filtrar por categoría: GET /recetas?categoria=postre
  • Filtrar por dificultad: GET /recetas?dificultad=baja
  • Búsqueda por nombre: GET /recetas?nombre=Tarta
